Adding Value, Subtract Headaches.

State starts in data-* attributes. $text="/count" binds the value to the DOM. @click="topic" delegates the event to a registered handler.

Inspect #counter in DevTools — watch data-count update.

HTML + JS 
customElements.whenDefined('data-wrapper').then(() => {
  const counter = document.getElementById('counter');
  counter.register({
    'count/inc': () => counter.put('count', n => Number(n) + 1),
    'count/dec': () => counter.put('count', n => Number(n) - 1),
  });
});
