Built-in formatters

Append ?formatter=arg to any $ binding to pipe the value through a transformer. Today the framework ships one built-in formatter; more will follow before 1.0.

NameSyntaxEffect
onoff ?onoff=truthy:falsy Picks one of two labels based on the bound value's truthiness. Bare ?onoff falls back to on/off.

Custom formatters

Register your own formatter at startup:

import { DW_FORMATTERS } from 'data-wrapper';

DW_FORMATTERS.set('currency', v =>
    new Intl.NumberFormat('en-US', {
        style: 'currency',
        currency: 'USD',
    }).format(Number(v) || 0)
);

// Use: <span $text="price?currency"></span>
