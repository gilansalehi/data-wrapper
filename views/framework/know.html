01 — recognize it

You already know this.

A variable. A button. One binding. That's a reactive component. 

<script type="module" data-module="@view/counter">
    export let count = 0;
    export const doubled = () => count * 2;
    export function increment(e) { count += Number(e.target.value); }
</script>

<button @click="increment?prevent" value="1">Increment</button>
<output $text="count"></output>
<output $text="doubled"></output>

No compiler touched that. The browser just ran it.
