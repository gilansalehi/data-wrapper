02 — write the view

Export values, then bind the HTML to them.

A view is regular HTML with a module script at the top. Exported values are readable by the markup, and exported functions can be called from browser events. 

<script type="module" data-module="@view/counter">
    export let count = 0;
    export function increment() { count += 1; }
</script>

<button @click="increment">Increment</button>
<output $text="count"></output>

The @click binding calls increment. After the function changes count, the $text binding reads it again and updates the output.
