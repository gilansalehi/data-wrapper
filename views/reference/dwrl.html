DWRL Reference

Data Wrapper Resource Locator. A URL-shaped addressing scheme parsed entirely by the browser's native new URL(). The token determines intent; DWRL names the resource.

Path syntax

SyntaxResolves to
/keyWrapper-root state key
/firstNameCamel-cased data-first-name / state.firstName
./keyItem-scoped (inside *list template only)
//other-id/keyCross-wrapper paths are reserved for a future release

Query params

ParamEffectExample
?format=name Apply a named formatter. Chain multiple — applied left to right. $text="/price?format=currency"
?key=field Override the identity key for *list reconciliation (default: id). *list="/users?key=uuid"
