Tokens

TokenDirectionWhat it doesExample
$ State -> DOM Binds a state path to a DOM property. Works on any property the element exposes. $text="/username"
* State -> DOM structure Runs a structural directive. Built-ins are *if and *list. *if="/ready", *list="/items"
@ Event -> handler Delegates a native DOM event to a registered handler topic. One listener per event type on the wrapper. @click="todo/remove"
