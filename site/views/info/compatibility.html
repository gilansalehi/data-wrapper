Browser Compatibility

data-wrapper targets modern evergreen browsers. The core runtime depends on native Custom Elements, HTML templates, ES modules, dynamic import(), fetch, URL, and URLSearchParams. Runtime import-map support is the newest requirement; the framework script supports an data-shim-src fallback for es-module-shims.

Compatibility references: MDN import maps, MDN custom elements, MDN dynamic import, and MDN templates.
