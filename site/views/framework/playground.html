05 — put it together

A small app is still just the three tokens.

The todo view combines value bindings, structural directives, and event actions in one file. There is no component compiler behind it; the browser loads the view, data-wrapper wires the attributes, and the exported functions update the connected DOM.

Open the source to see the actual file that powers the widget. The list rows use *list, row bindings use ./, and the buttons call exported functions with @.
