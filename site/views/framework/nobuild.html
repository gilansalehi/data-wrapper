01 — one script tag

Add the script to your page.

The script registers the <data-wrapper> element and the small runtime that loads views. You can try the first component without creating a project or running a build. 

<script src="https://unpkg.com/data-wrapper"></script>

Use the package from a CDN while prototyping, or serve the built file yourself when you want full control.

You do not need:

  • package.json
  • webpack
  • vite
  • JSX
  • a compiler
  • config files
