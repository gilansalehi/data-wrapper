08 — format and extend

Keep common display logic in the binding.

Formatters are small transforms that run after a value is read and before it reaches the DOM. They cover common presentation work like text cleanup, dates, money, booleans, sorting, joining, and JSON display.

Extension hooks

Add project-specific formatters with DW_FORMATTERS or structural behavior with DW_DIRECTIVES.

Explicit refresh

Use action() around writers and flush() when outside code changes state.

Small helpers

on, emit, p, and q are available when a view needs a little glue.
