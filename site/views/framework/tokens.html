03 — three tokens

Wire your HTML with $, @, and *.

A view exports values and functions from its module script. The HTML uses three attribute tokens to read those exports, call those functions, and render repeated or conditional markup.

The buttons call exported functions. After a function changes count, the $text binding reads the value again and updates the output. The source shown here is the real file used by the live component.

The three tokens:

$ write a value
<output $text="count"></output> Read an exported value and write it into text, attributes, properties, or classes.
@ call a function
<button @click="plus"></button> Listen for a browser event and call an exported function from the view.
* render markup
<template *list="todos"> Repeat a template with *list, or show a block with *if.
