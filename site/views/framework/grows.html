05 — compose views

Split bigger screens into smaller files.

When a page gets larger, keep the same model. A wrapper can load a layout view, and that layout can fill its own outlets with *src. The source can be a child view URL or authored content supplied by the host. 

<template *src="nav"></template>
<template *src="panel"></template>

The host writes the tabs once. The layout decides where they go and which view appears in the panel.
