Frameworks read across the columns, comparison axes down the rows. Uncheck frameworks to narrow the field — two side-by-side is often the most useful view. Each cell carries a letter grade for that axis: the grades are editorial ballparks, not benchmarks, and the No-build friendliness row deliberately scores how well each tool serves a ship-files workflow (so a required compiler grades low, however good the compiler is). Bundle sizes are omitted because they drift with every release — check bundlephobia for current numbers (data-wrapper's own build is ~7 kB min+gzip as of this writing).
