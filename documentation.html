Tokens

The wire dispatch. $prop sets el[prop] = value; *directive invokes a registered structural directive; @event delegates a native DOM event to a registered topic.

State API

Every mutation routes through put() and emits data:sync. patch shallow-merges, push appends, pull filters. register wires event topics to handlers.

Bindings ($)

$prop="/path" sets el[prop] = value on update. Most names map directly; the table below covers the HTML-attribute / DOM-property naming gap.

*list

Reconciles a child <template> against an array. Reuses rows by identity (default key id; override with ?key=field), broadcasts updated values into existing row subscribers, and renders a data-empty template when the array is empty.

*if

Toggles an element's presence in the DOM. When false, the element is replaced with a comment anchor; when true, it returns and re-wakes.

Empty-state templates

Built-in templates referenced by data-empty="name" on a *list element. Override by declaring a <template id> with the same name in your document.
